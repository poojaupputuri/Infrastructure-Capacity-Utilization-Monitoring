# Infrastructure-Capacity-Utilization-Monitoring
deployed link:https://capacity-monitor.vercel.app/
### The Challenge
City planners today face:
- 📊 **Spreadsheet overload** - Data scattered across multiple files
- 🗺️ **No visual context** - Can't see problems geographically
- ⚠️ **Delayed response** - Issues detected after they become critical
- 💰 **Wasted resources** - Underused assets go unnoticed
### Our Solution
**CityCapacity** provides a unified dashboard that:
- ✅ Monitors 31+ infrastructure assets in real-time
- ✅ Color-codes problems (⚫ overload, 🔴 congestion, 🔵 underuse)
- ✅ Alerts planners instantly when thresholds are crossed
- ✅ Covers 10 major Indian cities
